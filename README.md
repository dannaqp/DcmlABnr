# Descripción
Convierte números decimales de 0 a 99 en binario de 8 bits con ceros a la izquierda.  
Ejemplo: 13 → 00001101

# Instrucciones para ejecutar

1. Entrar al repositorio:  
   https://github.com/dannaqp/DcmlABnr.git

2. Hacer clic en Code (botón verde).

3. Hacer clic en Codespaces.

4. Hacer clic en Decimal a Binario.

5. En la barra lateral izquierda, dar clic en el botón Play (triángulo verde) y observar el resultado de la conversion.

6. Ingresar el número de dos cifras cuando aparezca el mensaje

7. Observar el resultado en la terminal
